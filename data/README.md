# Data README

## Purpose

This directory stores all project datasets split by lifecycle stage.

- raw: immutable source tables generated by the synthetic data generator
- interim: optional intermediate transforms
- processed: model-ready features and embeddings
- external: third-party inputs (currently empty)

## Data lineage

- data/raw/*.parquet is produced by src/falabella_risk/data/generate_data.py
- data/processed/features.parquet is produced by src/falabella_risk/features/feature_engineering.py
- data/processed/gnn_embeddings.parquet is produced by src/falabella_risk/models/train_gnn.py
